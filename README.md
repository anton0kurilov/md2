# Material Design 2.0 (Material Theme) CSS Framework
**Still in development!**
## Install

Include the line in your HTML page:
```
<link rel="stylesheet" type="text/css" href="https://rawgit.com/anton0kurilov/md2/master/md2.css">
```
Enjoy!
